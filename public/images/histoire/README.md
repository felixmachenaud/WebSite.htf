# Images de la page Histoire

Pour personnaliser la page Histoire, placez vos 4 photos ici et mettez à jour les chemins dans `app/a-propos/histoire/page.tsx` (const IMAGES).

| Fichier | Section |
|---------|---------|
| `hero.jpg` | Grande photo du hero (40–50 % hauteur visible) |
| `fondation.jpg` | Section « Une école née d'une conviction » |
| `vie-scolaire.jpg` | Photo vie scolaire / relation éducative |
| `activites.jpg` | Photo activités / sorties |

**Contraintes :** lumière naturelle, ambiance calme et institutionnelle, cohérence esthétique entre les 4 images.

Par défaut, la page utilise des images existantes du site (college, lycée).
